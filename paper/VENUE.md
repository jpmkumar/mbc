# Target Venue: IEEE Access (Primary)

**Status:** Selected as primary submission target  
**Backup:** Quantum Machine Intelligence (Springer)

## Why IEEE Access

- Rolling submission (no fixed deadline)
- Typical review: 4–6 weeks
- Accepts hybrid QML + medical AI interdisciplinary work
- Open access with broad visibility
- Encourages code/data availability statements

## Template

LaTeX manuscript: [`paper/main.tex`](main.tex)  
Uses standard `IEEEtran` document class (IEEE Access compatible).

## Submission Requirements Checklist

- [ ] Manuscript in IEEE format (10–12 pages typical)
- [ ] Abstract: 150–250 words, no abbreviations undefined
- [ ] Figures ≥300 DPI
- [ ] Code availability statement
- [ ] Ethics/data use statement (public datasets)
- [ ] Author ORCID and affiliations
- [ ] Supplementary material (hyperparameters, extended ablations)
- [ ] Submit via [IEEE Access Author Portal](https://ieeeaccess.ieee.org/)
