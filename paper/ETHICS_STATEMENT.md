# Ethics and Data Use Statement

This study uses publicly available de-identified medical imaging datasets:

- **CBIS-DDSM** (mammography) — The Cancer Imaging Archive
- **BUSI** (ultrasound) — Kaggle / Data in Brief
- **Breast Thermography Database** — public research datasets

No patient-identifiable information (PHI) is stored or processed in this repository. All experiments comply with the terms of use of each dataset provider. Synthetic data generation (`data/download/generate_synthetic.py`) is provided for pipeline validation only and does not represent real patient records.

For submission to IEEE Access, include this statement in the manuscript Data Availability / Ethics section.
